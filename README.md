# React Native Legend Calendar

An opinionated fork of Marcelo Prado's [flash-calendar](https://github.com/MarceloPrado/flash-calendar) library which leverages [legend-list](https://github.com/LegendApp/legend-list) instead of flash list and is heavily optimized.

I am not the original author and I deserve no credit, I am just iterating on Marcelo's awesome work with legend-list for my own use cases.

## Documentation & Examples

You can find the Legend Calendar documentation [on the website](https://leviwilliams.github.io/legend-calendar/).
